集成Charts,绘制多组柱状图随记

发表于 2019-01-18
1.集成Charts

​ 首先在github上下载Charts 打开工程，将Charts.xcodeproj和 Source,Tests文件拖到你自己的工程中去。

因Charts是swift编写的，本工程是oc编写，故需要进行oc和swift混编。在工程里随意创建一个swift文件，会弹出一个选项，点击 Create Bridging Header 此时，Xcode会自动创建一个桥接文件。你的工程名-Bridging-Header 在此文件中添加 @import Charts;

然后在工程target-general中选择Embedded Binaries中点击+ 添加Charts.framework

在build settings中将swift language version 中选择swift最新版。

此时在你需要的文件中导入桥接文件编译即可成功。
